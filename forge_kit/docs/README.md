# ForgeKit Documentation

ForgeKit is a structured game creation framework for Godot 4.x. The current release (v0.1.2) includes an RPG module with 14 resource types, a 5-tab editor dock, visual editors, and a Quick Setup system.

This documentation assumes you already know Godot basics.

## Getting Started

- [Quickstart](quickstart.md) - Install ForgeKit and generate your first set of resources in about 10 minutes.
